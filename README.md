# easy.math v.2.x.x

Библиотека математических функций easy.math.

Библиотека разрабатывалась для плееров версий QSP 5.7.0 - это классический плеер версии 5.7.0, плеер от Seedhartha для Android до версии 2.3.1, а так же Quest Navigator версии 0.0.28. Не гарантируется правильная работа на плеерах иных версий. Некоторые функции имеют ограничения на значение числовых аргументов, это связано с используемыми плеером библиотеками. Для разных версий плеера могут быть разные ограничения, поэтому внимательно читайте документацию к плееру, под который разрабатываете игру.
